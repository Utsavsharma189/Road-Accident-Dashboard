# Road-Accident-Dashboard
 1) Project Title / Headline

Road Accident Analysis Dashboard – Tableau (Safety & Pattern Insights)

 2) Short Description / Purpose

This project is an interactive Tableau dashboard designed to analyze road accident data and uncover meaningful insights related to accident severity, casualty distribution, vehicle involvement, and environmental conditions. The dashboard helps identify accident patterns across time, location, and contributing factors to support better understanding of road safety risks.

 3) Tech Stack
Tableau Desktop – Data visualization and dashboard development
Tableau Data Prep (or built-in cleaning) – Data preparation and transformation
Excel / CSV – Raw dataset storage and preprocessing
Kaggle Dataset – Source of road accident data

 4) Data Source
Dataset sourced from Kaggle
Contains structured road accident records including:
Accident severity (slight, serious, fatal)
Casualties involved
Vehicle types
Road surface and weather conditions
Lighting conditions
Time and location of accidents

 5) Features & Highlights
 Accident Severity Analysis
Breakdown of accidents by severity levels to understand impact distribution.
 Casualty Analysis
Tracks total casualties and severity-wise impact of road accidents.
 Vehicle Type Insights
Identifies which vehicle types are most frequently involved in accidents.
 Environmental Condition Analysis
Analyzes how weather, road surface, and lighting conditions affect accident occurrence.
 Location-Based Insights
Highlights accident-prone areas and high-risk zones.
 Time Trend Analysis
Examines accident patterns across hours, days, and seasonal trends.
 Interactive Dashboard Filtering
Enables dynamic exploration using filters and parameters.
 Safety Insights Generation
Converts raw accident data into actionable insights to support road safety improvements.
